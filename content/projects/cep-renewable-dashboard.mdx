## Overview

Collaborating with California Energy and Power (CE+P) to design and build a comprehensive, interactive dashboard that visualizes renewable energy data alongside a curated industry news feed. The platform aims to give stakeholders a real-time view into the renewable sector's performance, trends, and developments.

## Goals

- Build an interactive, real-time dashboard with filterable metrics across renewable energy categories (solar, wind, hydro, etc.)

- React / Next.js for the frontend application
- Data visualization libraries (D3.js, Recharts, or similar)
- Backend API for data ingestion and news aggregation
