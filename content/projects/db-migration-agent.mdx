## Architecture

- **Orchestrator Agent**: Manages the overall migration pipeline, delegates tasks, and handles exceptions
- **Schema Mapping Agent**: Analyzes source and target schemas, identifies field mappings, and flags incompatibilities
- **Data Transformation Agent**: Handles type conversions, data cleaning, and format normalization
- **Validation Agent**: Runs integrity checks, identifies data loss risks, and generates migration reports
